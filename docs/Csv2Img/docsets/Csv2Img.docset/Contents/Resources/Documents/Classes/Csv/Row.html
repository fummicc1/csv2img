public struct Row

Row (a line)

Row is hrizontally separated group except first line.

First line is treated as ColumnName.

eg.

1 2 3 4

5 6 7 8

→Row is [5, 6, 7, 8].

Because this class is usually initialized via Csv, you do not have to take care about Row in detail.
